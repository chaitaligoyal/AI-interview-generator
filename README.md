AI Resume Interview Assistant

An AI-powered interview preparation application built using Streamlit and OpenRouter APIs.
The app analyzes uploaded resumes and generates personalized technical and HR interview questions using Generative AI.

Features:

1.Upload Resume PDFs

2.AI-generated Technical Interview Questions

3.AI-generated HR Interview Questions

4.Resume Review and Analysis

5.Candidate Strength Analysis

6.Resume Improvement Suggestions

7.Resume Scoring System

8.Clean and Interactive Streamlit UI

9.Deployed Online using Streamlit Cloud


Tech Stack:

1.Python

2.Streamlit

3.OpenRouter API

4.OpenAI SDK

5.pypdf

6.python-dotenv


Deployment:

This project is deployed using Streamlit Community Cloud.

1.Generative AI workflows

2.Prompt Engineering

3.LLM API Integration

4.Streamlit Deployment

5.PDF Text Extraction

6.Environment Variable Management

7.AI-powered Document Analysis
